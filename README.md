## Overview

Document QnA System is an AI-powered Question Answering application that enables users to extract information from documents and interact with them using natural language queries. The project leverages Retrieval-Augmented Generation (RAG) to retrieve relevant document chunks and generate accurate responses using Large Language Models.

Currently, the system supports PDF-based question answering and serves as the foundation for future multimodal capabilities involving images and visual content.

---

## Features

* PDF document ingestion and processing
* Automatic text chunking for efficient retrieval
* Semantic search using vector embeddings
* FAISS vector database for fast similarity search
* Retrieval-Augmented Generation (RAG) pipeline
* Natural language question answering
* Google Gemini integration for response generation
* Modular architecture for future expansion
